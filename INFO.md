# Radio Player — Проект

## Коротко
Автономний настільний інтернет-радіоприймач на ESP32-S3.
Вмикаєш — грає. Без смартфона, без підписок, без складнощів.

---

## Апаратура

| Компонент | Деталі |
|---|---|
| Плата | Viewe UEDX80480043E-WB-A |
| МК | ESP32-S3 (dual-core 240MHz, 8MB PSRAM Octal) |
| Дисплей | 4.3" IPS 800×480 RGB, контролер ST7262E43 |
| Тач | GT911 (capacitive, I2C) |
| Аудіо | Зовнішній I2S DAC (моно) |
| Сховище | microSD |
| Мережа | WiFi 2.4GHz (вбудований) |

---

## Що має вийти в фіналі

### Поведінка пристрою
- Вмикається → одразу грає останню станцію на збереженій гучності
- Без підтверджень, без меню завантаження
- Стабільна робота 24/7
- Відновлення після втрати WiFi або живлення

### UI на екрані
- **Темна тема** — фон #09101A
- **Годинник** — SevenSeg 96px по центру, синє glow
- **Назва станції** — Roboto Medium 16px, uppercase, letter-spacing
- **Назва треку** — Roboto Light 15px, горизонтальний scroll якщо довга
- **Лічильник** — "X / N" поточна / всього станцій
- **Спектрограма** — жива canvas-анімація на фоні, вертикальний градієнт
- **Кнопки** — PREV / PLAY / NEXT в один рядок (110 + 238 + 110 px)
- **Гучність** — слайдер 20px з drag-badge SevenSeg, кнопка Mute 58×50px
- **Статусбар 54px** — ліворуч ☰ і ⚙, центр IP/статус, праворуч WiFi RSSI
- **Панель станцій** — swipe-up або ☰, повноекранна, список не закривається при виборі

### Сховище (SD карта)
Вся конфігурація у відкритих форматах — редагується прямо на SD через ПК:
```
/stations.json   — список станцій (назва, URL, жанр)
/config.json     — гучність, остання станція, налаштування
/www/            — файли веб-інтерфейсу
/logs/           — логи помилок
```
WiFi credentials — єдине що зберігається в NVS (безпечно, не на SD).

### Веб-інтерфейс (radio.local)
- Той самий темний дизайн, адаптивний
- Керування відтворенням
- Редагування списку станцій
- Імпорт/експорт M3U плейлистів
- Перевірка URL станцій
- OTA оновлення прошивки
- Системна інформація (uptime, RSSI, heap, версія)

## Технічний стек

| Шар | Технологія |
|---|---|
| Фреймворк | ESP-IDF v5.5.4 |
| UI | LVGL v9 |
| BSP | Viewe lvgl_v9_demo (esp_lvgl_port) |
| Аудіо | ESP-ADF: http_stream → mp3/aac decoder → i2s_stream |
| Формати | MP3, AAC (основні), Ogg Vorbis (якщо дозволить RAM) |
| Веб | ESP HTTP Server + WebSocket |
| DNS | mDNS (radio.local) |
| Час | SNTP (NTP синхронізація) |
| OTA | ESP HTTPS OTA |

### Прийняті архітектурні рішення (не переглядати)
- ESPHome — відхилено (надмірно для цього проекту)
- SquareLine Studio — не використовується (пишемо LVGL вручну)
- Arduino framework — відхилено (ESP-IDF дає повний контроль)
- SD карта — єдине сховище конфігурації (NVS тільки WiFi)
- Аудіо pipeline: `http_stream_reader → music_decoder → i2s_stream_writer`
