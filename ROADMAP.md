**Мета:** пристрій грає інтернет-радіо через I2S DAC.

- [ ] Підключити ESP-ADF або власний HTTP stream клієнт
- [ ] HTTP stream reader (підтримка Icecast/Shoutcast)

- [ ] I2S output на DAC
- [ ] Буферизація (unevenness мережі)

- [ ] Інтеграція з UI: статуси BUFFERING / CONNECTING / ERROR
- [ ] Обробка розриву з'єднання + автоперепідключення

## ⏳ Фаза 4 — SD карта + дані

**Мета:** конфігурація на SD, NTP час, WiFi provisioning.

- [ ] Монтування SD (SPI або SDMMC)
- [ ] Читання `stations.json` → список станцій в UI
- [ ] Читання `config.json` → гучність, остання станція
- [ ] Запис `config.json` при зміні (гучність, вибір станції)
- [ ] SNTP синхронізація → реальний час на годиннику
- [ ] WiFi підключення з NVS credentials
- [ ] WiFi provisioning (якщо credentials відсутні → onboarding)
- [ ] Логування помилок в `logs/`
